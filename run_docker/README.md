# Thai Text to Speech Docker

by docker :

```
docker pull wannaphong/thaitts
docker run -d -p 5000:5000 wannaphong/thaitts
```

use :

```
http://127.0.0.1:5000/tts?text=ทดสอบ ระบบ
```
